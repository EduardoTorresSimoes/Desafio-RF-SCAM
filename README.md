# Desafio RF-SCAM

CRUD MVC de relacionamento Empregado/Departamento utilizando Node.js, Express, PostgreSQL e Angular.

## Estrutura

- `backend/` — API REST com Node.js + Express
- `frontend/` — Angular

## Pré-requisitos

- Node.js 18+
- PostgreSQL
- Angular CLI

## Backend

```bash
cd backend
npm install
cp .env.example .env  # ajuste as variáveis
npm run dev
```

## Frontend

```bash
cd frontend
npm install
ng serve
```
